Exercicio 2

2. Crie uma página contendo uma textarea onde o usuário poderá digitar um comentário. Inclua dois inputs do tipo radio com seleção mutuamente exclusiva: as opções são importante e não importante. Caso o usuário escolha inserir um comentário importante, o texto deve ser escrito na cor vermelho; caso contrário, o texto deve ser inserido na cor azul. Os comentários devem ser parágrafos (tag p) inseridos após o botão utilizado para realizar a inserção.
